# Navier-Stokes Topological Depletion Framework

A novel stochastic Lagrangian approach using **exact Gauss linking numbers** as a depletion term in a Lyapunov functional.

- Preprint: `main.tex` (full paper)
- Lean 4 formalization: `bifurcation.lean` (100% complete bifurcation theorem)
- Simulation code: `simulation.py` (vortex-filament method)
- Plots: `/plots/` folder (Hou–Luo smoothness + extreme singularity runs)

This is exploratory research presenting a new bifurcation criterion for the 3D Navier-Stokes Millennium problem (smoothness OR singularity based on initial topology).
